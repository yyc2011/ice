## 模块说明

### 顶部导航栏
- 左侧「←」返回消息中心
- 中间显示对方昵称
- 右侧显示对方头像缩略图，点击跳转对方个人主页

### 消息气泡规则

| 方向 | 说明 |
|---|---|
| 对方消息 | 头像在右侧，气泡在左侧，左对齐 |
| 我的消息 | 头像在左侧，气泡在右侧，右对齐 |
| 时间分隔线 | 同一会话内相邻消息间隔超过 30 分钟，自动插入时间线 |

### 发送状态说明

| 发送状态 | 气泡变化 |
|---|---|
| 发送中 | 气泡旁显示加载圆圈 |
| 发送成功 | 正常展示 |
| 发送失败 | 气泡旁显示红色感叹号 ❗，点击重试 |

### 私信发送规则（来自 MC-2）

| 场景 | 输入框状态 | 额外提示 |
|---|---|---|
| 对方已关注我 | 正常可用 | 无 |
| 对方未关注我，未发过消息 | 正常可用 | 顶部提示「将消耗书币」+ 输入框下方显示书币余额 |
| 对方未关注我，已发 1~2 条未回复 | 正常可用 | 输入框上方显示「还可发送 N 条」（N = 3 − 已发条数） |
| 对方未关注我，已发 3 条未回复 | 禁用 | 「等待对方回复后可继续」 |
| 书币不足 | 禁用 | 「书币不足」；`recharge_enabled=true` 时展示充值入口，否则仅文案提示 |

> 书币单价已定（5书币/条），完整规则见 [`05-track-书币系统-2026-05-05.md`](05-track-书币系统-2026-05-05.md)。

### 本阶段仅支持文字
- 仅文字输入，不支持图片/表情包/语音（后续迭代再加）
- 单条消息无字数上限（暂定，后续可加）
