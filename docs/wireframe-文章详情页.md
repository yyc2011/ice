## 模块说明

### 顶部固定栏

| 元素 | 说明 |
|------|------|
| `←` 返回 | 返回上一页 |
| `♥ 点赞(N)` | 点击切换点赞/取消点赞状态；N 实时更新；**D5已定**：顶栏是点赞唯一入口，(⋯) 面板不重复 |
| `收藏` | **D3已定**：收藏入口位于顶部栏；点击切换收藏/取消收藏（已收藏态样式待视觉定） |
| `↑ 分享` | 点击后从底部弹出全宽分享面板，图标+文字横向排列三项：微信好友 / 朋友圈 / 生成图片；点「取消」或面板外区域关闭 |
| `(⋯)` 圆形选项按钮 | 点击后底部面板从下方弹出，覆盖屏幕下方约 1/3；再次点击面板向右滑出消失 |

### 正文区

- 字体、字号、行距与写文页保持一致（所见即所得基准）
- 图片以独立块形式换行显示
- 文章末尾展示标签胶囊

### 文字引用评论

- 用户长按正文选中文字后弹出浮动气泡（「复制」「引用评论」）
- 点击「引用评论」：页面自动滚动至文后评论区（状态4），底部输入框获得焦点，上方自动附带引用块
- 引用文字超过100字自动从第100字处截断，末尾加「...」
- 引用块：深灰色背景、字号小于正常评论正文、无任何前缀文字；右上角 `×` 可删除，删除后变为普通评论

### 选项面板（状态3）

- **P0 已定**：评论列表已移至状态4（文后内联），本面板只做操作区
- 从底部弹出，覆盖屏幕下方约 1/3 高度，上方 2/3 为半透明遮罩
- 点击遮罩区域或再次点击 `(⋯)` → 面板向右滑出
- **D1已定**：面板操作项为「差评 · 更多」，不含评论列表
- **D2已定**：举报入口收入「更多」弹层，点击「更多」后展开「🚩 举报此文章」
- **D5已定**：点赞在顶栏已有，面板不重复放置；差评为面板主要操作项

| 操作项 | 说明 |
|--------|------|
| 差评(N) | 点击后弹出书币确认弹窗（状态5）；消耗 5 书币/次，不退还；余额不足引导充值 |
| [ ··· 更多 ] | 点击展开更多操作；当前仅含「🚩 举报此文章」 |
| 🚩 举报此文章 | 单选原因（7 项，见 10-track 附录 B）；选「其他」须填说明；确认后消耗 10 书币，举报成立退还 |

### 评论列表规则（状态4 文后内联）

| 元素 | 说明 |
|------|------|
| 一级评论 | 大头像靠左；昵称在头像右侧同行顶部；评论正文在昵称下方与其左对齐；底部行：`时间 · 地区 · 回复`靠左，`♥N 差评`靠右；时间格式：24h内用相对时间（刚刚/N分钟前/N小时前），超过24h用绝对日期（2026-04-20 18:10:23） |
| 引用块 | 显示在评论正文上方；深灰色背景；字号小于正文；无前缀文字 |
| 二级评论 | 在一级评论正文左边缘处缩进；小头像靠左；`昵称`（若回复二级则追加 `▶ 回复 目标昵称`）同行；回复正文下一行；底部行格式同一级 |
| 展开/折叠 | 默认展示前3条二级；超出显示「展开 N 条回复 ∨」；展开后末尾显示「收起 ^」 |

### 评论输入框

- **唯一位置**：固定在页面底部（状态4文后内联），**不再出现在状态3面板中**
- 左侧输入框，占位文字「期待你的评论...」；右侧三个图标：📷（图片）/ @（艾特）/ 😊（表情）
- 点击输入框后键盘弹起，输入框随之上移
- 有引用时（长按正文→引用评论触发）：输入框上方显示引用块（深灰色背景，小字号，`×` 可删除）
