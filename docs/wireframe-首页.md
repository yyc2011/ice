## 底部 Tab 说明

| Tab | 说明 |
|---|---|
| 首页 | 当前页 |
| 话题 | 跳转话题总览页，对所有话题分类展示 |
| ✏️（凸起） | 进入写文页，视觉突出，鼓励创作 |
| 分类 | 跳转管理员两级分类全目录页（点小类进文章列表；与话题是两个独立概念） |
| 我的 | 个人主页、书币、消息等 |

## 模块说明

### 模块1 热门话题
- **H2已定**：横向滑动卡片，首页固定展示 **5 个**进行中话题
- **排序已定**：`status=进行中`，按 `article_count` 降序，同分按 `start_at` 降序；取 Top5
- 每张卡片显示：封面图、话题名称、参与文章数
- 点击卡片 → 跳转话题详情页
- 模块右下角/末尾固定「查看全部话题 →」入口，点击 → 跳转话题页

### 模块2 24小时热榜
- **H1已定**：展示 **Top5**（第1-5名），共5条，不再跳第10名
- 点击「查看完整榜单」→ 跳转完整榜单页
- 完整榜单页支持 `24小时 / 7天 / 30天` 三个Tab切换，默认24小时
- **排序已定**：过去 N 天内**发布**的文章，按 `hot_score` 降序（详见 [`10-track` 附录 A](10-track-需求文档-v1-2026-07-07.md)）；24h Tab 与书币每日 Top10 结算同源

### 模块3 精选新文
- 固定展示 10 篇
- **排序已定**：近 7 天发布，`featured_score = ai_quality×0.7 + freshness×0.3`（详见 10-track 附录 A）
- 文章卡片中的作者头像/昵称可点击，跳转对应个人主页
- 点击「查看更多新文」→ 跳转 [`wireframe-精选新文列表页.md`](wireframe-精选新文列表页.md)

### 模块4 按分类浏览
- **G7已定**：固定展示 **5 个推荐大类**，由后台配置决定展示哪5个，前端不做动态排序或懒加载
- 每个大类展示 **5 篇**文章预览（该大类近 30 天 `hot_score` Top5）
- 每个大类底部有「查看更多 XX →」按钮，点击 → 跳转分类页并定位到对应大类区块
- 模块最底部固定一个「进入全部分类 →」按钮，点击 → 跳转分类页（底部 Tab「分类」对应页）
- 本模块不再支持"向下滚动加载更多分类"，分类全目录入口由分类 Tab 承担

### 模块5 关注动态
- **H3已定**：关注功能本阶段不做，本模块暂不展示，下阶段迭代再设计
- 占位标注保留在线框中，提醒后续接入时此处为预留模块位置

### 顶部搜索栏
- 默认按文章标题搜索
- 点击右侧「▾」可切换搜索条件：标题 / 作者 / 标签 / 话题
- 返回规则：标题/标签仅返回文章；作者仅返回用户；话题仅返回话题
- 搜索后跳转 → 搜索结果页
