## 底部 Tab 说明

消息中心无独立 Tab。入口位于「我的」Tab 页右上角消息图标（🔔）：
- **角标规则**（MC-3 已定）：仅统计私信未读数；互动消息和系统通知的未读角标仅在消息中心内显示，不计入「我的」Tab 角标

## 模块说明

### 统一会话列表排序规则
- **互动消息**、**系统通知**、**私信会话**三类条目统一按「最后消息/互动时间」倒序混排（MC-1 已定）
- 互动消息条目的时间 = 最新一条互动（点赞/评论/关注）的时间
- 系统通知条目的时间 = 最新一条系统通知的时间

### 会话条目结构

| 字段 | 私信会话 | 互动消息 / 系统通知 |
|---|---|---|
| 左侧图标 | 对方用户头像（圆形） | 固定系统图标（🔔 / 📢） |
| 标题行 | 对方昵称 | 固定名称「互动消息」/「系统通知」 |
| 摘要行 | 最后一条消息内容截断 | 最新一条通知摘要 |
| 时间 | 最后消息时间，遵循 G2 规范 | 最新通知时间，遵循 G2 规范 |
| 角标 | 该会话未读消息数 | 该类型未读条数（超99显示「99+」） |

### 点击行为

| 点击目标 | 跳转页面 |
|---|---|
| 私信会话条目 | 私信会话页（wireframe 待开） |
| 「互动消息」条目 | 互动消息列表页（wireframe 待开）|
| 「系统通知」条目 | 系统通知列表页（wireframe 待开）|
| [✏️] 发起新私信 | 用户搜索页，选人后进入会话 |

长按私信会话条目 → 出现操作菜单：「删除会话」（本地删除，不通知对方）

### 私信发送规则（MC-2 已定）

> 完整私信 UI 待「私信会话页」线框时展开，此处仅记录业务规则供后续参考。

| 条件 | 规则 |
|---|---|
| 对方已关注自己 | 免费发送，无条数限制 |
| 对方未关注自己，且对方尚未回复过 | 消耗书币发送（5书币/条）；最多发 **3 条**，超过 3 条后发送按钮禁用，显示引导文案 |
| 对方未关注自己，但对方已回复过 | 消耗书币发送（5书币/条）；条数不再限制（对话已激活） |

**第 3 条已发送、等待回复时的提示文案**（发送框禁用状态）：  
`「已发送3条消息，等待对方回复后即可继续」`

**书币不足时**：发送按钮点击后提示余额不足 + 跳转书币充值入口（单价：5书币/条，完整规则见 `05-track-书币系统-2026-05-05.md`）

## 已定决策汇总

| 编号 | 决策 | 结论 |
|---|---|---|
| MC-1 | 系统条目排序 | 混入私信列表按最新时间统一排序，不置顶 |
| MC-2 | 私信发起范围 | 全站用户均可发；对方未回复最多3条；对方未关注自己时消耗书币（5书币/条）；对方回复后正常私信 |
| MC-3 | 「我的」Tab 角标 | 仅计私信未读数；系统消息角标仅在消息中心页内显示 |
