## 模块说明

### 顶部导航栏
- 左侧「← 返回」：返回上一页（话题页 Tab 或首页话题模块）
- 右侧「分享」：点击后从底部弹出全宽分享面板（状态2）；内容与文章详情页分享面板一致：微信好友 / 朋友圈 / 生成图片；点「取消」或面板外区域关闭

### 模块1 话题头部
- 状态徽章取值：进行中（绿色）/ 即将开始（蓝色）/ 已结束（灰色）
- 进行中时显示截止日期；即将开始时显示开始日期；已结束时显示结束日期
- **TD3已定**：标题下方显示聚合统计行：`📝 参与篇数  👁 总阅读量  ♥ 总点赞数`；数据由后端接口返回

### 模块2 素材区
- 仅有标题的话题：整个模块2不展示
- 有图片素材：图片在上（16:9比例），文字素材在下
- 仅有文字素材（无图片）：直接显示文字，无图片占位
- 文字素材较长时：默认折叠显示约4行，底部显示「展开全部 ↓」
- 展开后：显示完整文字，底部改为「收起 ↑」

### 模块3 参与
- 话题状态为「进行中」时：显示「写这个话题」按钮
- 话题状态为「即将开始」或「已结束」时：隐藏按钮，仅显示统计行
- 点击「写这个话题」→ 需登录；跳转写文页，话题信息已预填
- **TD3已定**：参与区统计行与模块1一致，显示 `📝 参与篇数  👁 总阅读量  ♥ 总点赞数`
- **TD1已定**：投票模块本阶段不做，下阶段再设计；线框中以占位注记标注

### 模块4 话题文章
- 默认选中「热度榜」Tab
- 热度榜排序：**排序已定** — 话题内已发布文章复用全站 `hot_score`（见 10-track 附录 A）；进行中实时计算，结束后取 `end_at` 时刻快照
- 最新排序：按文章发布时间倒序
- 每条文章显示：排名序号（热度榜）/ 作者名称 / 文章标题 / 阅读量 / 点赞数
- 点击文章 → 跳转文章详情页
- 无限滚动加载，无分页按钮
