## 模块说明

### 设置项（M1 已定）
| 设置项 | 说明 |
|---|---|
| 我的账号 | 展示用户登录名；90 天内限改 1 次；未设置时提示在设密时一并设置 |
| 手机号绑定/更换 | 短信验证 |
| 微信号 | 只读展示（微信登录绑定） |
| 修改登录密码 | 首次须设账号 + 密码两次确认；修改时验证原密码 |
| 退出登录 | 清除登录态，跳转登录页 |
| 账号注销 | 危险操作，二次确认 + 输入确认文字 |

### 跳转关系
- 入口：我的页「设置 →」
- 「←」返回我的页
