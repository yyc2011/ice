## 一、范围

### 本批做

| 模块 | 内容 |
|---|---|
| 本地上传 | `POST /api/v1/uploads/images` 落盘 `./data/uploads/{yyyy}/{MM}/`；`GET /api/v1/uploads/files/**` 公开读（日后换 OSS 只换存储实现） |
| 热榜/精选 | DTO 扩字段；精选 `page/size`；小程序完整页 |
| 话题 | 详情 + `/{id}/articles`；发起话题（封面上传/书币/审核）；写文 `topic_id` 预填；Lv2 入口拦截 |
| 搜索 | `GET /api/v1/search?type=title\|author\|tag\|topic`；小程序搜索结果页 |

### 本批不做

- 真实 OSS、个人主页、书币充值页、「我发起的话题」列表页（toast 占位）
- 话题投票、搜索历史、生成分享图、朋友圈分享
- 互动赞评藏（P2）

## 二、上传目录约定

| 项 | 值 |
|---|---|
| 配置 | `ice.upload.local-dir` 默认 `./data/uploads` |
| 相对路径 | `{yyyy}/{MM}/{uuid}.{jpg\|png\|webp}` |
| 公开 URL | `{scheme}://{host}/api/v1/uploads/files/{相对路径}` |
| 忽略 | `server/data/uploads/`、`data/uploads/` 已入 `.gitignore` |

小程序上传走 `miniprogram/api/upload.ts`（`wx.uploadFile`），不依赖 OpenAPI FormData。

---

## 三、接口补齐摘要

| 方法 | 路径 | 说明 |
|---|---|---|
| POST | `/api/v1/uploads/images` | 需登录；返回 `{ url }` |
| GET | `/api/v1/uploads/files/**` | 公开 |
| GET | `/api/v1/articles/featured` | `?page&size` → `{ items, total, page, size }` |
| GET | `/api/v1/topics/{id}/articles` | `?sort=hot\|latest&page&size` |
| GET | `/api/v1/search` | `?q&type=title\|author\|tag\|topic` |

鉴权公开读：`search`、`topics/{id}`、`topics/{id}/articles`、`uploads/files/**`（与 featured/rankings 一致）。
