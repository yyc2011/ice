## 一、范围

### 本批做

| 能力 | 说明 |
|---|---|
| 手机号 + 短信验证码 | 登录/注册、已登录绑定/换绑 |
| 登录密码 | 设置/修改、手机号+密码登录 |
| 微信真实资料 | `wx.getUserProfile` → 同步昵称/头像 |
| 账号注销 | 软删除/匿名化 |
| 设置页 | 对应 wireframe-设置页 |

### 本批不做

- 未登录只读浏览首页（继续 G1 全站需登录）
- 微信号独立解绑（小程序内 `openid` 天然绑定；注销时清空 `openid`）
- 邮箱登录

## 三、接口契约

### `POST /api/v1/auth/sms/code`

**请求**

```json
{ "phone": "13800138000", "purpose": "login" }
```

**响应 200**

```json
{ "message": "验证码已发送", "expires_in": 300 }
```

限流：同手机号 60s 一次；每日上限 10 次。

### `POST /api/v1/auth/phone/login`

**请求**

```json
{ "phone": "13800138000", "code": "123456" }
```

**响应 200**：同 `wechat/login`（`token` + `user`）。

用户不存在则创建（`nickname` 默认「手机用户」+ 后缀）。

### `POST /api/v1/auth/password/login`

**请求**

```json
{ "account_name": "my_account", "password": "******" }
```

按用户自定义 `account_name` 登录（非 `internal_uid`、非手机号）。

### `POST /api/v1/me/phone/bind`（需登录）

**请求**

```json
{ "phone": "13800138000", "code": "123456" }
```

### `POST /api/v1/me/password`（需登录）

**请求**

```json
{
  "account_name": "my_account",
  "old_password": null,
  "new_password": "******",
  "confirm_password": "******"
}
```

首次设置：`old_password` 可空，须同时传 `account_name`；修改时 `old_password` 必填；`confirm_password` 必须与 `new_password` 一致。

### `POST /api/v1/me/account-name`（需登录）

**请求**

```json
{ "new_account_name": "new_name", "password": "******" }
```

修改用户登录名；90 天内限 1 次；须校验登录密码。

### `GET /api/v1/me/account`（需登录）

**响应 200** 扩展字段：

```json
{
  "phone_masked": "138****1234",
  "phone_verified": true,
  "password_set": true,
  "wechat_bound": true,
  "account_name": "my_account",
  "account_name_set": true,
  "can_change_account_name": true,
  "account_name_next_change_at": null
}
```

不返回 `internal_uid`。

### `POST /api/v1/me/wechat-profile`（需登录）

**请求**

```json
{ "nickname": "张三", "avatar_url": "https://..." }
```

### `DELETE /api/v1/me`（需登录）

软删除：`status=1`（封禁）、清空 `openid`/`phone`/`password_hash`/`account_name`/`nickname` 改为「已注销用户」、`avatar_url` 清空；`internal_uid` 保留。文章保持原 `status` 不动。

## 三.1 账号字段说明（V13）

| 字段 | 说明 |
|------|------|
| `internal_uid` | 15 位随机数字；创建时生成；全局唯一、不可改；用户不可见 |
| `account_name` | 用户自定义登录名；`[a-zA-Z0-9_]` ≤15 位；首次设密时设置；90 天内可改 1 次 |

## 四、短信服务

| 环境变量 | 说明 |
|---|---|
| `SMS_MOCK_ENABLED` | `true` 时固定验证码 `123456`，日志打印 |

## 五、错误码

| code | HTTP | 说明 |
|---|---|---|
| `SMS_SEND_TOO_FREQUENT` | 429 | 发送过于频繁 |
| `SMS_CODE_INVALID` | 400 | 验证码错误或过期 |
| `PHONE_ALREADY_BOUND` | 409 | 手机号已被其他账号绑定 |
| `PASSWORD_LOGIN_NOT_SET` | 400 | 未设置密码 |
| `PASSWORD_INCORRECT` | 400 | 密码错误 |
| `PASSWORD_MISMATCH` | 400 | 两次密码不一致 |
| `ACCOUNT_NAME_REQUIRED` | 400 | 首次设密未填账号 |
| `ACCOUNT_NAME_INVALID` | 400 | 账号格式不合法 |
| `ACCOUNT_NAME_TAKEN` | 409 | 账号已被占用 |
| `ACCOUNT_NAME_CHANGE_LIMIT` | 400 | 90 天内已修改过账号 |
| `ACCOUNT_NAME_NOT_SET` | 400 | 账号未设置或不存在 |

## 六、小程序

### 设置页 `pages/settings`

- 我的账号（展示/修改；90 天限改 1 次）
- 手机号绑定/更换（短信验证弹窗）
- 登录密码设置/修改（首次须设账号 + 密码两次确认）
- 退出登录
- 账号注销（二次确认 + 输入「确认注销」）

### 我的页

- 资料卡旁「授权头像昵称」按钮 → `wx.getUserProfile` → `POST /me/wechat-profile`
- 「设置 →」入口

### 取舍说明

**不做微信号解绑**：产品运行于微信小程序，`openid` 与小程序账号一一对应；独立解绑与注销语义冲突。设置页仅展示「微信号：已绑定（微信登录）」只读状态。
