## 模块说明

### 模块1 顶部栏
- 「←」返回我的页
- 「充值 →」跳转书币充值页（**仅当** `config.client.features.recharge_enabled=true` 时展示；否则隐藏）

### 模块2 余额区
- 展示当前书币余额（大字）

### 模块3 流水列表
- Tab 筛选：全部 / 收入 / 支出
- 每条记录：类型名称、关联内容摘要、金额（+/-）、时间
- 按时间倒序，分页滚动加载
- 时间格式遵循 G2 规范

### 流水类型（示例）
| 类型 | 方向 |
|---|---|
| 微信充值 | 收入 |
| 发表文章奖励 | 收入 |
| 热榜奖励 | 收入 |
| 话题奖励池到账 | 收入 |
| 发起话题 | 支出 |
| 话题奖励池充值 | 支出 |
| 差评 | 支出 |
| 举报 | 支出 |
| 申请复审 | 支出 |
| 发送私信 | 支出 |
| 字数突破 | 支出 |
| 查看访客列表 | 支出 |

### 跳转关系
- 入口：我的页「书币明细 →」
- 去向：书币充值页、返回我的页
