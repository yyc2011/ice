## 模块说明

### 入口与权限
- 入口：话题页右下角「+ 发起话题」FAB
- 权限：Lv2 成长及以上或管理员；不足 Lv2 在话题页即被拦截，不进入本页

### 表单字段
| 字段 | 必填 | 说明 |
|---|---|---|
| 话题标题 | 是 | 最多 20 字 |
| 话题描述 | 否 | 最多 200 字 |
| 持续时间 | 是 | 7 / 14 / 30 天三选一 |
| 封面图 | 否 | 无则使用默认纯色+标题 |
| 奖励池 | 否 | 最低 50 书币；创建时从余额锁定扣除 |

### 书币扣减
- 发起话题固定消耗 **20 书币**（`cost_start_topic`）
- 奖励池金额另计，创建时一并锁定扣除
- 审核失败：发起费用 + 奖励池**全部退还**
- 审核通过：不退还
- 管理员强制下架：奖励池退还发起人

### 跳转关系
- 入口：话题页 FAB（权限通过后）
- 「←」返回话题页
- 余额不足 → 书币充值页
- 提交成功 → 返回话题页或我的页「我发起的话题」
