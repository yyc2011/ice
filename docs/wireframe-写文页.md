## 模块说明

### 顶部操作栏

| 元素 | 说明 |
|------|------|
| `←` 返回 | 普通箭头；有未保存内容时弹确认对话框："是否保存草稿？"三个选项：保存草稿 / 不保存退出 / 取消 |
| `[保存草稿]` | 顶部栏居中；手动保存当前内容为草稿，不受字数上限限制；保存成功后顶部短暂提示"已保存" |
| `N/M字` | 位于 `[保存草稿]` 右侧、`(⋯)` 左侧；M 为当前等级免费上限 + 本篇已购突破字数；达免费上限时数字变红，正文禁止新增输入（可删改）；草稿保存不受限 |
| `(⋯)` 圆形按钮 | 展开发布设置面板（全屏覆盖，从右上角动画展开）；再次点击收缩回编辑页；按钮始终悬浮可见 |

### 正文编辑区

- 所见即所得：写文页字体、字号、行距与文章详情阅读页保持一致
- 纯文字输入，无格式工具栏
- 插图以独立块形式新起一行显示，不可嵌入行内

### 字数统计

- 显示位置：顶部固定栏，`[保存草稿]` 右侧、`(⋯)` 左侧，格式为 `N/M字`
- 任何时候实时更新，图片不计入字数
- M（上限）= 当前等级免费上限 + 本篇已通过书币购买的额外字数；Lv5 用户达 5000 字后不可再突破
- 达到当前实际上限时：数字变红，正文禁止新增输入；底部展示书币突破提示条（状态3）
- **W4已定（修订）**：达免费上限时展示「消耗10书币/+500字」入口；可多次购买直至 5000 字全站上限；规则见 [`05-track-书币系统-2026-05-05.md`](05-track-书币系统-2026-05-05.md)，等级上限见 [`06-track-等级系统-2026-05-20.md`](06-track-等级系统-2026-05-20.md)
- 发布设置面板中「发布文章」：仅当正文字数 ≤ 当前实际上限时可用

### 发布设置面板

| 设置项 | 是否必填 | 说明 |
|--------|----------|------|
| 标签 | 必填 | 搜索框实时搜索已有标签；有匹配时展示结果列表可点选；无匹配时出现「＋ 新增标签」，点击新增并自动选中；已选以胶囊展示可删除；最多5个；未填时「发布文章」置灰 |
| 分类 | 选填 | 模糊搜索跨大类和小类；默认展示大类折叠列表，点击展开小类；选中后以「大类 > 小类」格式显示，可点 × 清除；用户不可新建分类；未选则发布后自动归入默认分类，用户后续可修改 |
| 关联话题 | 可选 | 搜索框实时搜索已有话题；默认展示进行中的话题列表；无匹配时出现「＋ 新增话题」，点击新增并自动选中；选中后以胶囊展示可取消；单选 |
| 插图 | 可选 | 最多3张；上传后图片插入到**打开面板前光标所在位置的下一行**；若光标在标题行或未曾定位，则兜底**追加到正文末尾**（W2已定）；可在此处删除已插入图片 |
| 封面图 | 可选 | **W1已定**：有插图时自动取第一张作封面（默认行为，无需操作）；无插图时显示默认占位图；「自定义封面」开关默认关闭，开启后可上传另一张图替换当前封面 |

> **插图交互说明**：用户先在正文中将光标定位到想插入图片的位置，再点击 `(⋯)` 打开面板上传。面板记录打开时的光标位置，图片上传完成后自动插入到该位置的下一行。  
> **W2已定·插图兜底行为**：若打开面板前光标停留在标题行、或从未定位过任何正文位置，则图片兜底**追加到正文末尾**。

> **背景音乐**：本阶段不做，发布面板中无 BGM 入口（P0 已定，后续阶段再设计）。  
> **W3已定·申请人工审核入口**：该功能本阶段不做，发布面板中无此入口，后续再考虑。
